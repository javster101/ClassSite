Ready to register?

Are you ready to start or boost your kid's programming journey? Sign up now!
Requirements are minimal. We will be using Zoom, a free service, to host the classes. To code, we use Repl.it online, so no downloads are needed. All that is needed for this class is a computer with internet access (Chromebooks work) and a microphone. We highly recommend having a web camera to ensure engagement, but if you do not feel comfortable with that please just let us know beforehand.

Sessions

We offer the following sessions for both paths:

  • July 13th-July 17th
  • July 20th-July 24th
  • July 27th-July 31st

Future sessions are subject to interest, so come back occasionally to see if more weeks are available!

Pricing

We offer competitive pricing for all options!

  • Introductory class half day: $150
  • Introductory class full day: $240
  • High School/AP class half day: $200
  • High School/AP class full day: $350

We run different projects each week, so if you would like to sign up your kids for multiple weeks, we will independently tailor the teaching section to run advanced concepts for them.
In addition, we offer 10% discounts for subsequent weeks if you sign up the same student for two or more weeks! Please contact Javier (coindreau.j@northeastern.edu) if you would like to take advantage of this deal.
(If you are doing this, please do not pay through the registration form)

Refund Policy

As our satisfaction guarantee, we offer full refunds up to the end of the first day of classes. If you would like a refund, please contact us as soon as possible to ensure that we are able to process your refund.
